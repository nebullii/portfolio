# Public Assets

This folder contains public assets for your portfolio.

## Files to Replace

- **`me.webp`** - Replace with your own profile photo
- **`og-image.png`** - Replace with your own Open Graph image (1200x630px recommended)
- **`favicon.svg`** / **`favicon.ico`** - Replace with your own favicon
- **`YourName_Resume.pdf`** - Add your resume PDF file (referenced in index.astro and Footer.astro)

## Optional Files

- **`your_certificate.jpg`** - Add your certification images if you want to display them
- **`preview.png`** / **`globe_preview.png`** - Preview images for documentation

## Image Guidelines

- Profile photo (`me.webp`): Use WebP format for best performance, square aspect ratio recommended
- OG Image (`og-image.png`): 1200x630px for optimal social media sharing
- Favicon: SVG preferred for scalability, with ICO fallback for older browsers
